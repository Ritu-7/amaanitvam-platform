# Amaanitvam Platform

Official technology platform of Amaanitvam Foundation.

## Overview

Amaanitvam Platform is the centralized digital platform for the foundation's initiatives, programs, community engagement, certificate verification, and future digital services.

This repository contains the complete source code, documentation, and engineering assets for the platform.

## Technology Stack

### Frontend

* HTML5
* Tailwind CSS
* JavaScript (ES Modules)
* Vite

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL
